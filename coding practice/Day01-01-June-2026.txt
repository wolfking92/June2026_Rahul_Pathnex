Ansible Task - Install Nginx on Pathnx Server 

---
- name: Install Nginx on Pathex Server
  hosts: all 
  become: yes

  tasks:
    - name: Install nginx
      yum:
        name: nginx
        state: present

    name: nginx

Jenkins Pipeline — Checkout Git

pipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'https://github.com/Pathnex/sample-java-app.git'
            }
        }
        stage('List Files') {
            steps {
                sh 'ls -la'
            }
        }
    }
}






GitLab CI — Checkout Git
